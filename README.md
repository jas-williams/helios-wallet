# Helios Wallet
![Demo](https://s2.gifyu.com/images/sample_vid.gif)

## Install dependencies
```
cd helios-wallet
yarn install
```

## Development
```
yarn develop
```

## Build from source


```
yarn pack:mac
yarn pack:win
yarn pack:linux
```
